Description:

Fully functional web implementation of the Ticket to Ride board game, themed after Marvel's Infinity war. Project was completed by a team of 5 students over the course of a semester using an Agile workflow. Project incorperates many Object Oriented design patterns and a Model View View Model (MVVM) UI architechture.  A fully playable version of the project can be accessed from the above link.

Features:

Angular front end

Kotlin server back end

Angular Material UI componants

Command Pattern game play serialization 

Multiple data permenance options (SQL, Flat File) via Plugin Pattern

Scalable vector graphics

In-game chat and game history

Automatic reentry into games after lost connection

Longest Route compuation
