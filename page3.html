Description:

GrandpARents is an AR-based system that helps elderly people with visual and/or hearing impairments to cope with their daily tasks. There are two main features in our system. The first one is real-time subtitle for one-on-one conversations; the second feature is Zoom, which allows the user to zoom in a selected area. GrandpARents was also tested via a useability test to verify the usability and functionality of our system. The project was completed by six students over the couse of a quarter using an Agile workflow.

Features:

Real Time conversation captions

Static image capture and magnificaiton

Augmented Reality interface via Microsoft HoloLens

Developed using the Unity framework and MRTK

Simiple UI 

Azure speech to text integration
